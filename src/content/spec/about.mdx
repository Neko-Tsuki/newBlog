## 🚀 从 Hexo 到 Astro (The Journey)

经过一个月的“大搬家”，我正式从 Hexo 迁徙到了 **Astro**。

**为什么是 Astro？**

并不是 Hexo 不够优秀，而是 Astro 给了我一种“全知全能”的错觉。在这里，没有预设的束缚，每一个页面、每一行 CSS 都可以完全自定义。在折腾主题的过程中，我不小心点满了许多~~邪道~~前端技能点。虽然它不像 Hexo 那样“开箱即用”，但这种 **亲手构建** 的快感正是开发者所追求的。
